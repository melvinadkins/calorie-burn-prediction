# Raw Dataset

The original dataset originates from the Kaggle Playground Series S5E5 competition:

https://kaggle.com/competitions/playground-series-s5e5

Due to repository size constraints, the raw CSV files are not stored in this repository.

To reproduce results:

1. Download `train.csv` and `test.csv` from Kaggle.
2. Place them inside `data/original/`.
3. Run the notebooks or training pipeline.
